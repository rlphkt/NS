READ ME

Dit is mijn prototypen voor project 2, dit project moest ik een verhalen website maken voor de Ns. Deze verhalen worden op de overzichtspagina weergegeven. Op de overzichtspagina kan de gebruiker zijn gelijk het boek vinden waar hij mee bezig is, ook zal hij op de hoofdpagina boeken vinden die uitgelicht worden. Alle boeken zijn opgedeelt in 3 catogerieÃ«n. De gebruiker kan een eigen account aanmaken om verhalen op te slaan. Deze verhalen kunnen ook gedownload worden zodat hij deze offline kan lezen of luisteren. Uit mijn onderzoek kwam namelijk dat het luisteren van contect erg fijn is voor de gebruiker, hierdoor kunnen zij zich namelijk focussen op meer dan alleen hun telefoon.

De html is opgebouwd in met een body, header, main en een  footer. Deze verschillende tags hebben daarin weer andere html tags zodat ik de verhalen kan weergeven. Ook maak ik gebruik van verschillende wrappers om mijn pagina samen te stellen.


Met javascript heb ik geprobeert om de micro interacties nog interactiever temaken. Deze bestanden zijn in de map javascript te vinden.
